Create charts from your Django models effortlessly

Do you need to plot charts in your django websites? Are you using boring static chart images and do you like your charts to be more interactive and interesting? Do you have to write a lot of boilerplate code just to convert the data to the right format before you can plot? Django-Chartit helps to ease the pain of creating interactive charts on the webpages. Charts are rendered using Highcharts JS library and jQuery. Get started with the demo!
